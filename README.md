## Setup

Install [meteor](https://www.meteor.com/install).

## Dependencies

```
meteor npm install
```

## Database

```
meteor mongo
```

## Running

```
meteor

```

## Testing

```
meteor test --driver-package practicalmeteor:mocha
```
